Pet Clinic management system:
#Aplication

    + The user will interact with the system via command line interface where he/she can choose an option.
    + The desktop application will not have any visual friendly frames


#Vet
    + As a user i should be able to add a new veterinarian, i should be able to specify firstname, lastname,
      address, speciality
    + As a user i should be able to view the details of an existing veterinarian, i should be able to view the
      following data :  id, firstname, lastname,address, speciality
    + As a user I should be able to update all existing veterinarian by choosing the id, i should be able to edit
      firstname, lastname, address, speciality
    + As a user i should be able to delete an existing veterinarian by choosing the id
    + As a user i should be able to view a list of all veterinarians, i should be able to view the following data:
      id, firstname, lastname

#Pet
    + As a user i should be able to add a new pet, i should be able to specify race, birthdate, isVaccinated, ownerName
    + As a user i should be able to view the details of an existing pet, i should be able to view the
          following data :  id, race, birthdate, isVaccinated, ownerName
    + As a user I should be able to update all existing pets by choosing the id, i should be able to edit
          race, birthdate, isVaccinated, ownerName
    + As a user i should be able to delete an existing pet by choosing the id
    + As a user i should be able to view a list of all pets, i should be able to view the following data:
          id, race, ownerName

#Consult
    + As a user i should be able to add a consult for a pet, i should be able to specify: vet Id of an existing,
      pet Id of an existing pet, date, description and an id.
    + As a user I should be able to view a list of all consults, i should be able to view the following data :
      consult Id, vet Id of an existing, pet Id of an existing pet, date and description
    + As a user I should be able to update the description for a specific consult by choosing the id.

#Import Data

    + As a user i should be able to import all the data from .csv file collection

#Optional

    + As a user i should be able to authenticate before using the application
